=== Wordpress Admin Dashboard Widget ===
Contributors: Bogere Goldsoft
Tags: reactjs, wordpress plugin development

Why ?
Just see how Reactjs can be integrated into the wordpress plugin development.
That makes us able to build versatile admin dashboard because reactjs is widely 
for large scale apps. 

== Installation ==
	1. Upload the entire `seo_dash` folder to the `/wp-content/plugins/` directory.
	2. Activate the plugin through the 'Plugins' menu in WordPress.

== Development ==
	- After installation, for the development purposes go ahead install the nodejs modules
	using this command... `npm install`
	- then go ahead and run `npm start`
	you will find Recharts Admin Dashboard widget
